```🚌 SafeTransit  Smart Public Transport Safety```

An AI-powered public transport safety platform designed to protect commuters through real-time alerts, anonymous incident reporting, and live journey tracking.

📱 About the Project
SafeTransit is a mobile-first web app UI concept built for the IDEATHON 2026 competition. It addresses the critical need for safety in public transportation by empowering commuters with smart tools to stay safe, report incidents, and share their journey in real time.

✨ Features

🆘 SOS Button — One-tap emergency alert that notifies trusted contacts and authorities instantly
🗺️ Safe Route Planner — AI-suggested routes based on real-time incident data and lighting conditions
📍 Live Journey Tracker — Share your live location with family/friends during travel
⏱️ Auto-Alert System — Automatically fires an alert if you don't confirm safe arrival
🔒 Anonymous Incident Reporting — Report harassment, unsafe drivers, or suspicious activity with zero identity exposure
📊 Authority Dashboard — Real-time heatmap and incident stats for transport control authorities


🛠️ Built With

CSS3 (custom properties, animations, glassmorphism)

🎯 Problem Statement
Public transport users — especially women and late-night commuters — face safety risks with no easy way to raise alerts, report incidents, or verify route safety. SafeTransit bridges this gap with a smart, accessible, and privacy-first solution.
